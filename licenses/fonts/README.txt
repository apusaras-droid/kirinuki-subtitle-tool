Japanese subtitle fonts installed by setup.bat

Source:
https://github.com/google/fonts

Pinned source commit:
389b770410cc0b7c21c85673bfa2077420fe7f65

Families:
- Noto Sans JP
- M PLUS Rounded 1c
- Dela Gothic One
- BIZ UDPGothic
- Zen Kaku Gothic New
- Zen Maru Gothic
- Zen Old Mincho

The installer downloads each family's OFL.txt into this directory. M PLUS
Rounded 1c uses the OFL.txt pinned from the official M PLUS source repository.
The font files are installed for the current Windows user and are not stored in this repository.
