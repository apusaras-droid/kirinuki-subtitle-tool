# エフェクト3分類管理

このアプリの装飾エフェクトは、処理対象と実装方式で3分類する。
分類を混ぜるとプレビュー、最終出力、バグ調査の責務が曖昧になるため、新規エフェクトは必ず下記のどれかに分類する。

## 1. 動画加工

元動画のピクセルを加工するエフェクト。

主な実装先:

- ブラウザプレビュー: WebGL/GLSL または CSS filter
- 480p/最終出力: FFmpeg filter

例:

- 色調補正
- モノクロ
- セピア
- ハーフトーン
- 魚眼
- ブラー
- RGBずれ
- グリッチ
- 手ぶれ
- VHS/CRT
- ポスタライズ
- スポットライト
- 丸絞り暗転

注意:

- 字幕、吹き出し、文字枠は加工対象にしない。
- 最終出力では、先に元動画へ動画加工をかけ、その後で字幕/枠/重ねる効果を合成する。

## 2. 重ねる効果

元動画の上に透明レイヤーとして追加するエフェクト。

主な実装先:

- ブラウザプレビュー: canvas/WebGL overlay またはDOM overlay
- 480p/最終出力: Python/Pillowで透過PNGまたは透過連番を生成し、FFmpeg overlayで合成

例:

- 集中線
- ハート
- 星
- 雪
- 風船
- 漫画効果線
- 画面端の飾り
- 衝撃フラッシュ
- シネマ帯
- 流れ星
- ハート漂い
- ハート枠拡大

注意:

- 形、線、ランダム感、中央の空き、透過を重視する効果はこの分類にする。
- FFmpeg式だけで無理に描画しない。Python生成レイヤーの方が調整しやすい。

## 3. 字幕/枠

字幕テキスト、吹き出し、文字に連動する装飾。

主な実装先:

- テキスト: ASS字幕またはPillow生成
- 枠: Python/Pillowで透過画像生成
- 文字連動: ASSイベントまたは個別オーバーレイ

例:

- 文字プリセット
- フォント
- 文字色
- 文字縁
- 吹き出し枠
- ナレーション枠
- 文字に反応するハートや星

注意:

- 画面全体の動画加工とは別管理にする。
- 個別字幕イベントの設定は、全体設定より優先できる。

## 現在のUI分類

- `テキスト`: 字幕/枠分類のテキスト部分
- `枠`: 字幕/枠分類の枠部分
- `文字連動`: 字幕/枠分類の文字反応部分
- `画面効果`: 動画加工と重ねる効果をカテゴリフィルターで分離

## 実装判断

新規エフェクトを追加するときは、最初に次の問いで分類する。

1. 元動画の色や形を変えるか。
   - はい: 動画加工
2. 元動画に何かを足すだけか。
   - はい: 重ねる効果
3. 字幕や吹き出しにだけ関係するか。
   - はい: 字幕/枠

迷った場合は、重ねる効果として別レイヤー化する方が安全。
