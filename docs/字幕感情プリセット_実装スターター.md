# 字幕感情プリセット 実装スターター

この文書は、字幕コメントとシーンの感情に応じたプリセットを先に実装するための最小構成です。

方針は次の通りです。

- 先に字幕の見た目を安定させる
- 映像エフェクトは別管理にする
- シーン単位とコメント単位を分ける
- 1つの感情に 1 つの既定プリセットを割り当てる

## 1. 先に作るもの

最初に実装するのは次の 4 つです。

1. `subtitle_style_preset`
2. `emotion_preset`
3. `scene`
4. `subtitle_item`

この順で進めると、字幕の見た目とシーン管理だけ先に安定させられます。

## 2. 管理対象

### 2.1 subtitle_style_preset

字幕の見た目を決めます。

主な項目:

- フォント名
- フォントサイズ
- フォントの太さ
- 文字色
- 縁取り色
- 縁取り幅
- 吹き出しの形
- 吹き出しの塗りつぶし色
- 吹き出しの枠色
- 影の有無

### 2.2 emotion_preset

感情ラベルから字幕スタイルを引くための入口です。

主な項目:

- 感情名
- 対象範囲
- 参照する字幕スタイル
- 参照するエフェクトグループ

### 2.3 scene

シーン単位の管理です。

主な項目:

- 開始時間
- 終了時間
- 感情ラベル
- 適用プリセット
- 対象コメント一覧

### 2.4 subtitle_item

1 行または 1 コメント単位の字幕です。

主な項目:

- 開始時間
- 終了時間
- 本文
- 話者ラベル
- 感情ラベル
- シーン ID
- 個別 override

## 3. 推奨ファイル構成

```text
project/
  presets/
    emotion_presets.json
    subtitle_style_presets.json
  scenes/
    scenes.json
  subtitles/
    subtitles.json
```

## 6. 適用ルール

優先順位は次の通りです。

1. 全体設定
2. シーン設定
3. コメント単位 override

同じ項目が複数レイヤーにある場合は、後勝ちにします。

## 7. この段階でやらないこと

- 映像エフェクトの本格実装
- シーンごとの複雑な合成
- 1コメントごとの高度な動的演出

先に字幕と感情の切り替えを安定させ、その後にエフェクトを足します。

## 8. 次の段階

字幕側が安定したら、次に `effect_group` を追加します。

そのときは、字幕スタイルとは別管理にして、問題があった場合に字幕側だけ先に直せる構造を維持します。
