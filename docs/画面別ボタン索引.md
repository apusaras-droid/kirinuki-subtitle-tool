## 1. 編集画面

### 読み込み・保存

- `動画を選択`
- `読み込み`
- `プロジェクト保存`
- `上書き保存`
- `字幕を保存`

保存先:

- `project.json`
- `edit_plan.json`
- `subtitles/edited.srt`
- `transcript/transcript.json`
- `temp/logs/*`

### 解析・編集

- `動画情報取得`
- `音声を抽出`
- `文字起こしを実行`
- `無音区間を検出`
- `カット案を作成`

保存先:

- `source/input.*`
- `audio/source_range.wav`
- `transcript/transcript.json`
- `subtitles/original.srt`
- `edit_plan.json`

### プレビュー・出力

- `仮出力を作成`
- `手動カットを仮出力`
- `最終出力`

保存先:

- `preview/preview_low.mp4`
- `preview/preview.srt`
- `output/final.mp4` または `output/final.mkv`
- `output/final.srt`
- `output/edit_plan_final.json`

### 位置調整

- `開始位置に設定`
- `終了位置に設定`
- `現在の開始/終了を区間に反映`

保存先:

- `project.json` の UI 状態
- `edit_plan.json` の `source_range`

### 波形編集

- `カット開始`
- `カット終了`
- `選択解除`
- `最後を削除`

保存先:

- `edit_plan.json.manual_cut_segments`
- `temp/logs/*`

### 画面切替

- `編集画面`
- `字幕編集`
- `プロジェクト一覧`
- `設定画面`
- `デコレーション画面`

- 装飾編集とは別工程にし、完成見込みの確認だけを行う。
