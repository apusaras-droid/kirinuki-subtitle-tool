# MPVエンコード無しプレビュー

## 目的

480pプレビュー動画をエンコードせず、元動画をMPVで開き、ASS字幕を外部字幕として読み込ませて即時確認する。

## 現在プレビューできるもの

- 元動画の再生
- ASS字幕として表現できる文字
- フォント、文字サイズ、文字色
- 文字の縁、縁色、影
- 字幕の表示開始・終了時刻
- 字幕位置

## 現在プレビューできないもの

- 無音カット後の完成タイムライン
- Pythonで生成する吹き出し枠PNG
- Pythonで生成するハート、集中線などの透過オーバーレイ動画
- FFmpegフィルタで処理する画面エフェクト
- 最終合成後の完全な見た目

## 理由

MPVのエンコード無しプレビューは、元動画に外部字幕を重ねるだけなので高速。
一方で、このアプリの枠や画面エフェクトはFFmpeg合成またはPython生成オーバーレイを前提にしているため、完全再現にはレンダー工程が必要。

## 使い分け

- 文字、フォント、字幕位置、タイミング確認: MPV即時プレビュー
- 枠、画面エフェクト、最終見た目確認: 480pプレビュー作成
- カット後の流れ確認: 仮出力または完成予定プレビュー
