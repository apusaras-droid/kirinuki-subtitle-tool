# 切り抜き字幕作成ツール 仕様準拠チェックリスト

## 1. 入出力

- [ ] ローカル動画を読み込める
- [ ] 対応拡張子は `mp4`, `mkv`, `mov`, `webm`
- [ ] `FFprobe` で動画情報を取得できる
- [ ] 指定区間を `HH:MM:SS.mmm` で指定できる
- [ ] 音声を WAV に抽出できる
- [ ] 元動画を破壊的に編集しない

## 2. 文字起こし

- [ ] `whisper.cpp` で文字起こしできる
- [ ] `whisper.cpp` を既定で選べる
- [ ] `openai-whisper` を選べる
- [ ] `faster-whisper` を選べる
- [ ] Whisper と VAD の処理を分離できる
- [ ] 声抽出音声を VAD 用に使える
- [ ] 声抽出音声を Whisper 用に切り替えられる
- [ ] 話者分離を使って字幕ごとに `speaker_label` を付けられる
- [ ] `speaker_label` を編集画面で変更できる
- [ ] `speaker_diarization_enabled` を設定できる
- [ ] `speaker_diarization_engine` を設定できる
- [ ] `speaker_diarization_engine` の既定を `speechbrain` にできる
- [ ] `min_speakers` を設定できる
- [ ] `max_speakers` を設定できる
- [ ] `output_audio_mode` を設定できる
- [ ] `detection_mode` を設定できる
- [ ] `pre_margin_sec` を設定できる
- [ ] `post_margin_sec` を設定できる
- [ ] `min_speech_duration_sec` を設定できる
- [ ] `min_silence_duration_sec` を設定できる
- [ ] `merge_silence_gap_sec` を設定できる
- [ ] 補正前と補正後のタイムスタンプを保持できる
- [ ] `transcript.json` を保存できる
- [ ] `original.srt` を保存できる

## 3. 無音削除

- [ ] 無音区間を検出できる
- [ ] Silero VAD で speech_start / speech_end の区間一覧を作成できる
- [ ] 発話前後の余白を設定できる
- [ ] 発話区間を結合できる
- [ ] keep_segments を作成できる
- [ ] SRT を編集後タイムラインへ再マッピングできる
- [ ] `edit_plan.json` を保存できる
- [ ] `source_video` はプロジェクト内相対で保存される

## 4. 編集

- [ ] 字幕一覧を表示できる
- [ ] 字幕本文を編集できる
- [ ] 字幕の開始・終了時間を編集できる
- [ ] 字幕を無効化できる
- [ ] `edited.srt` を保存できる
- [ ] `edit_plan.json` を更新できる

## 5. プレビューと出力

- [ ] 元動画プレビューができる
- [ ] 完成予定疑似プレビューができる
- [ ] 仮出力ができる
- [ ] 最終出力ができる
- [ ] `final.mp4` を出力できる
- [ ] `final.srt` を出力できる

## 6. 監査と配布

- [ ] API リクエストの監査ログが残る
- [ ] CLI 実行の監査ログが残る
- [ ] 外部コマンドの実行ログが残る
- [ ] CLI スモークテストで一括パイプラインを検証できる
- [ ] `project.json` は相対パス保存である
- [ ] `edit_plan.json` は相対パス保存である
- [ ] 配布用スクリプトで release を再現できる
- [ ] GPLv3 配布仕様書がある

## 7. CLI

- [ ] `python -m backend` で CLI を起動できる
- [ ] `probe` を実行できる
- [ ] `new-project` を実行できる
- [ ] `extract-audio` を実行できる
- [ ] `transcribe` を実行できる
- [ ] `detect-silence` を実行できる
- [ ] `create-edit-plan` を実行できる
- [ ] `preview` を実行できる
- [ ] `export` を実行できる
- [ ] `run-pipeline` で一括実行できる
- [ ] `run-pipeline --config` で JSON 設定を受けられる
- [ ] `report.json` を出力できる
- [ ] `scripts/smoke-test.ps1` を実行できる
