## 更新規則

- 画面順や工程状態を変えたら、工程分割UI適用方針と実装フローチャートを更新する。
- GUIボタン、API、CLI引数を変えたら、ボタンCLI索引と外部CLI連携を更新する。
- 保存形式や出力形式を変えたら、README、AI向け引き継ぎ、仕様準拠チェックリストを更新する。
- 配布物や依存関係を変えたら、配布仕様書とライセンス通知を更新する。
- 実装済みの判定はテストまたは実機確認を根拠にする。
