# 精度モード5分検証 問題切り分け

## 対象

- 検証プロジェクト: `5__20260717_2675d879`
- 対象範囲: 先頭300秒
- Whisper: whisper.cpp `large-v3` / Vulkan / AMD Radeon RX 7800 XT
- 声抽出: Demucs `htdemucs`
- VAD: Silero、しきい値0.32
- 字幕補正: VAD hybrid

## 結論

主原因はVADやWhisperの単体精度ではなく、`apply_vad_subtitle_corrections()`の対応付けと結合方法である。

1. Whisper生segmentの長い`whisper_start_sec/end_sec`をVADマッチングに使っている。
2. SRT向けに制限済みの`output_start_sec/end_sec`を無視している。
3. 同じVAD区間へ対応した複数字幕を無条件で1字幕へ結合している。
4. hybridがWhisperとVADの外側境界を採用するため、異常に長いWhisper時刻を再び字幕へ戻している。
5. VAD補正後に字幕表示時間の上限を再適用していない。

このため、補正前67字幕が補正後44字幕となり、36字幕が13個のVADグループへ結合された。最長例は「おやすみなさい」の約27秒表示である。

## 外部要因

### Whisper

Whisperのsegment時刻は近似値であり、無音や音楽のある区間では幻覚や長いsegmentが発生し得る。OpenAI実装も`no_speech_threshold`、`clip_timestamps`、`hallucination_silence_threshold`を持つ。したがって生segment境界を字幕表示境界として無条件に復元してはいけない。

### VAD

Silero VADは音声を「発話チャンク」へ分けるもので、1チャンクが1字幕を意味しない。公式実装の既定しきい値は0.5で、データセットごとの調整が推奨されている。

今回の分離音声による比較:

| threshold | 区間数 | 発話総秒数 | 最初の検出 |
|---:|---:|---:|---:|
| 0.32 | 57 | 152.9 | 21.2秒 |
| 0.50 | 56 | 142.1 | 21.2秒 |
| 0.65 | 61 | 130.1 | 35.0秒 |

0.32は精度モードとして低すぎ、歌声や分離残響を発話として拾う範囲を広げている可能性がある。

### Demucs

Demucsの`vocals`は会話専用の抽出ではなく、歌声を含むボーカルを伴奏から分離する機能である。アニメの主題歌や劇中歌は残るため、「声抽出済み = 台詞だけ」と扱ってはいけない。

今回の比較:

- 元音声VAD: 55区間、128.3秒
- 声抽出VAD: 60区間、162.3秒

声抽出は静かな台詞を回収できる一方、発話候補を約34秒増やした。

## 修正順序

1. Whisper字幕のIDと分割を維持し、VADインデックス単位の字幕結合を廃止する。
2. VADマッチングには`output_start_sec/end_sec`を使い、生の`whisper_*`は監査情報としてのみ保持する。
3. VADは字幕境界を近距離でクリップするために使い、WhisperとVADの外側を無制限に広げない。
4. 補正量、VADとの距離、重なり率に上限を設け、条件外は補正前時刻へフォールバックする。
5. 補正後に`normalize_subtitle_durations()`を再適用する。
6. BGM分離プリセットのVADしきい値を0.50前後へ上げ、0.45/0.50/0.60を素材別に比較できるようにする。
7. 最高モードでは、正しい字幕本文をWhisperXの日本語wav2vec2モデルでforced alignmentし、VADは前処理・無音抑制に限定する。

## 2026-07-17 実施結果

- VAD単位で複数字幕を結合する処理を廃止し、Whisper字幕のIDと本文単位を維持するよう変更した。
- VAD照合はWhisperの生時刻ではなく、正規化済みの `output_start_sec` / `output_end_sec` を使用する。
- VAD境界の補正量を開始・終了とも最大0.75秒に制限し、照合距離を最大0.5秒、最小重複率を0.1とした。
- 補正後に異常な長時間表示と字幕同士の重なりを再正規化する。
- 「精度・BGM分離」「最高・精度最優先」のSilero VAD閾値を0.50へ変更した。
- 既存5分データでは、字幕数は生字幕と同じ67件を維持し、10秒超の字幕は旧6件から0件、最大表示時間は4.5秒になった。
- 比較用SRTは `projects/5__20260717_2675d879/subtitles/aligned_vad_fixed_threshold_050.srt` に保存した。
